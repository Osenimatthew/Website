Key Features

Feature : Intuitive Controls

To move around the maze you can use this buttons W,A,D,S. the W button is to move forward, The S is to move bacwards, the a button is to turn to the left and the D button is to turn to the right. Other features will be added as we update the game

About This Project

This project was inspired by our passion for game development and the challenge of implementing raycasting techniques in C. We chose C because it was part of the requirement. Our team collaborated via GitHub and communicated through WhatsApp to ensure smooth progress and overcome challenges such as SDL initialization issues and raycasting bugs.
